üíª How to Use

With R (SQLite)

library(DBI)
library(RSQLite)

# Connect to database
con <- dbConnect(RSQLite::SQLite(), "eurlex_research_database.sqlite")

# View available tables
dbListTables(con)

# Query documents
documents <- dbGetQuery(con, "SELECT * FROM documents LIMIT 10")

# Search for specific terms
climate_docs <- dbGetQuery(con,
"SELECT celex_id, title, word_count FROM documents
WHERE full_text LIKE '%climate%' AND scrape_status = 'success'")

With Python (CSV)

import pandas as pd

# Load the dataset
df = pd.read_csv('eurlex_complete_dataset.csv')

# Basic exploration
print(df.shape)
print(df.columns)
print(df['document_type'].value_counts())

# Text analysis
climate_docs = df[df['full_text'].str.contains('climate', case=False, na=False)]

üìã Dataset Schema

Documents Table

  • celex_id - Unique EU document identifier
  • title - Document title
  • document_type - Type (Regulation, Directive, etc.)
  • date_published - Publication date
  • author - Authoring institution
  • full_text - Complete document content
  • word_count - Number of words in document
  • url - Original EUR-Lex URL

üìö Research Applications

  • Legal Research: Analyze EU legal frameworks and evolution
  • Policy Analysis: Track policy development across domains
  • Text Mining: Natural language processing of legal texts
  • Comparative Studies: Compare regulations across different areas
  • Temporal Analysis: Study how EU law has changed over time
